Setup & Execution Guide

Launch the horizontally scaled Chat Application locally.

1

Prerequisites

Ensure you have Docker and Docker Compose installed on your machine. If you are on Windows or Mac, downloading Docker Desktop is the easiest way to fulfill this. Ensure the Docker engine is running before proceeding.
2

Open the Project Directory

Open your terminal (PowerShell, Command Prompt, or Git Bash) and navigate to the project directory where the docker-compose.yml file is located.
cd desktop\devops\chat-app
3

Run the Multi-Container Architecture

To build the images and spin up all services in the background, run the following command. The --scale flag dynamically spins up multiple chat server instances. By default, let's start with 3 servers!
docker-compose up -d --scale chat-server=3
Note: The initial run will take a few minutes as it downloads official images for Nginx, Redis, MongoDB, and builds your custom Node.js image.
4

Access the Application

Once all containers state "Started", open your web browser and navigate to:
http://localhost
Open multiple tabs or use incognito mode to simulate different users. Notice the Server ID indicator in the top right to see Nginx's load balancer actively distributing your requests across the different replicas!
5

Stopping the Application

When you are finished testing, gracefully stop and remove the containers, networks, and images (except volumes) by running:
docker-compose down
If you want to clear your MongoDB and Redis database data as well, append -v to the command.
